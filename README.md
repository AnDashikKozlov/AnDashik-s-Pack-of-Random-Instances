# AnDashik's Pack of Random Instances
Just a pack of my custom (and modified VALVe's) instances for Portal 2 mapping.

Installation:
- Place folder `andashik_s_instances`, which contains instances, inside `Portal 2\sdk_content\maps\instances`;
- Place folder `Prefabs` inside `Portal 2\bin`;
- Optionally you can place bts_wall_module_zoo inside `Portal 2\sdk_content\maps`.

Includes:
- 10 512x512 BTS Wall Modules;
- 5 512x1024 BTS Wall Modules;
- A few additional prefabs instances, like 128_fuse_set01 with glow;
- VALVe's light instances modified with simple glow sprites.
